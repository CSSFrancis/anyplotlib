anyplotlib Documentation#

anyplotlib is a lightweight, interactive plotting library for Jupyter notebooks and JupyterLab, backed by anywidget and a pure-JavaScript canvas renderer. It follows the object-oriented Matplotlib API — create a Figure, call methods on Axes — while delivering real-time interactivity and high performance on large datasets through canvas-based blitting instead of SVG.
