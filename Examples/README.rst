Examples
========

A collection of short, self-contained examples demonstrating the
:class:`~viewer.viewer1d.Viewer1D` and :class:`~viewer.viewer2d.Viewer2D`
interactive widgets.

Each example is a plain Python script following the `Sphinx Gallery
<https://sphinx-gallery.github.io/>`_ format:

* The module-level docstring becomes the rendered narrative.
* ``# %%`` comment lines create new narrative / code cells.
* The script runs top-to-bottom, so every code block is fully executable.

Run any example directly::

    python Examples/plot_viewer1d_basics.py
    python Examples/plot_viewer2d_basics.py

Or open them as notebooks in Jupyter after converting with ``jupytext``::

    jupytext --to notebook Examples/plot_viewer1d_basics.py
